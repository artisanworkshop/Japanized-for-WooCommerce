# 特定商取引法表記

[← ドキュメントトップ](index.md)

---

## 概要

日本の「特定商取引に関する法律（特定商取引法）」では、通信販売を行う事業者は販売ページに所定の事項を表示することが義務付けられています。Japanized for WooCommerce では、特商法表記に必要な項目を管理画面で管理し、ショートコードでページに表示できます。

---

## 設定方法

**WooCommerce → JP4WC Settings → 特商法設定 (Law)** から各項目を入力します。

| 設定項目 | 内容 |
|----------|------|
| ショップ名 | 運営しているショップの名称 |
| 販売事業者名 | 法人名または個人事業主名 |
| 代表者名 | 法人代表者または個人事業主の氏名 |
| 店舗責任者名 | 通販責任者名 |
| 住所 | 事業所の所在地 |
| 問い合わせ窓口 | 問い合わせ方法（メールアドレス・フォーム URL など） |
| 電話番号 | 問い合わせ用電話番号 |
| 商品の価格 | 価格表示方法・消費税の説明 |
| 支払い方法・時期 | 使用できる支払い方法と支払いタイミング |
| 商品の購入方法 | 購入手順の説明 |
| 商品の引き渡し時期 | 発送・配送にかかる日数の目安 |
| 商品代金以外の料金 | 送料・代引き手数料・その他手数料 |
| 返品・キャンセル | 返品・交換・キャンセルの条件と手順 |
| 特別条件 | その他の特記事項 |

---

## ページへの表示方法

### 1. 特商法表記ページの作成

WordPress 管理画面の **固定ページ → 新規追加** から「特定商取引法に基づく表記」などのタイトルでページを作成します。

### 2. ショートコードの挿入

ページの本文に以下のショートコードを挿入します。

```
[jp4wc_law]
```

### 3. 表示確認

ショートコードが挿入されたページには、設定した内容が自動的に表形式で表示されます。

---

## 表示例

ショートコードは以下のような HTML テーブルを生成します。

| 項目 | 内容 |
|------|------|
| 販売事業者名 | 〇〇株式会社 |
| 代表者名 | 山田 太郎 |
| 住所 | 東京都〇〇区〇〇 1-2-3 |
| 電話番号 | 03-XXXX-XXXX |
| 支払い方法 | クレジットカード / 銀行振込 / 代金引換 |
| … | … |

---

## 注意事項

- 特定商取引法の要件は法改正等により変更される場合があります。最新の法令・ガイドラインを確認の上、適切な内容を記載してください。
- 法的な内容については専門家（弁護士・行政書士等）にご相談ください。
- 入力した内容はプラグイン側で `wp_kses()` によるフィルタリングが行われたうえでページに出力されます。一部の基本的な HTML タグ（例：`<strong>`, `<em>`, `<a>`, `<br>`, `<ul>`, `<ol>`, `<li>` など）のみ使用できますが、それ以外のタグや属性は削除される場合があります。
