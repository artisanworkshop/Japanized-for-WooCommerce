# 代引き・代引き手数料

[← ドキュメントトップ](index.md)

---

## 概要

商品の配達時に配送業者に代金を支払う「代金引換」決済方法です。WooCommerce 標準の代引きとは独立したゲートウェイとして提供されており、**代引き手数料の自動加算**と **WooCommerce Subscriptions（定期購入）との併用**に対応しています。

---

## 有効化

**WooCommerce → JP4WC Settings → 決済設定 (Payment)** の「代引き」セクションで有効にします。

---

## 代引きゲートウェイの設定

**WooCommerce → 設定 → 決済** から「代引き（JP4WC）」を選択して詳細設定を行います。

| 設定項目 | 説明 |
|----------|------|
| 有効化 | 決済方法を有効/無効にします |
| 表示名 | チェックアウト画面での決済方法名（例: 代金引換） |
| 説明文 | チェックアウト画面での説明 |
| 案内文 | サンクスページ・注文確認メールに表示する案内文 |
| 対応配送方法 | 代引きを利用できる配送方法を指定（空欄で全配送方法対応） |
| 仮想商品への対応 | 仮想商品のみの注文でも代引きを表示するか |

---

## 代引き手数料

### 機能概要

顧客が代引きを選択した際に自動的に手数料をカートに追加します。手数料額・税金計算・適用上限金額を設定できます。

### 設定方法

**WooCommerce → JP4WC Settings → 決済設定 (Payment)** の「代引き手数料」セクションで設定します。

| 設定項目 | 説明 |
|----------|------|
| 手数料名 | チェックアウト・注文詳細に表示される手数料の名称（例: 代金引換手数料） |
| 手数料額 | 加算する手数料の金額（円） |
| 手数料適用上限金額 | この金額未満の注文にのみ手数料を加算（0で制限なし） |
| 税金の計算 | 手数料に対して消費税を計算するか |
| 税区分 | 手数料に適用する税率の区分 |

### 手数料の表示

手数料は以下の場所に自動的に表示されます：

- チェックアウトページの注文サマリー
- サンクスページの注文詳細
- 注文確認メール
- 管理画面の注文詳細

---

## WooCommerce Subscriptions との併用

定期購入プラグイン「WooCommerce Subscriptions」を使用している場合、代引き（JP4WC）ゲートウェイは定期購入商品の決済にも対応しています。WooCommerce 標準の代引きは定期購入に非対応のため、定期購入商品を代引きで販売する場合はこちらのゲートウェイをご利用ください。

---

## ブロックチェックアウト対応

WooCommerce ブロックチェックアウトにも対応しており、追加設定なしで利用できます。

---

## 運用フロー

1. 顧客がチェックアウトで「代金引換」を選択して注文
2. 注文ステータスが「処理中」に設定される
3. 設定した代引き手数料が自動加算される
4. 注文を梱包・出荷し、配送業者に代引き設定で依頼
5. 配送業者が代金を回収し、店舗に送金
