# アフィリエイト連携

[← ドキュメントトップ](index.md)

---

## 概要

日本の主要なアフィリエイトサービスとの連携機能です。注文完了時に自動でコンバージョントラッキングタグを出力します。現在 A8.net と Felmat（フェルマ）に対応しています。

> **対応通貨**: 日本円（JPY）のみ

---

## 設定画面

**WooCommerce → JP4WC Settings → アフィリエイト (Affiliate)** から設定します。

---

## A8.net 連携

### 概要

日本最大級のアフィリエイトサービス A8.net のコンバージョントラッキングに対応しています。

### 設定方法

| 設定項目 | 説明 |
|----------|------|
| A8.net 連携 | 連携の有効/無効 |
| プログラム ID（PID） | A8.net の広告主管理画面で確認できるプログラム ID |
| テストモード | 有効にすると A8.net 公式のテスト PID を使用してトラッキングを確認できます |

### 動作

注文完了ページ（サンクスページ）に A8.net のコンバージョントラッキングタグが自動出力されます。

タグには以下の情報が自動的に含まれます：

- プログラム ID
- 注文ID（成果 ID）
- 注文金額
- 商品 ID / SKU / バリエーション ID
- 商品数量

### テストモードについて

テストモードを有効にすると、A8.net 公式のテスト用 PID を使用してタグの出力を確認できます。本番運用前に必ずテストモードで動作確認を行ってください。

---

## Felmat（フェルマ）連携

### 概要

アフィリエイトサービス Felmat のコンバージョントラッキングに対応しています。

### 設定方法

| 設定項目 | 説明 |
|----------|------|
| Felmat 連携 | 連携の有効/無効 |
| プログラム ID（PID） | Felmat の管理画面で確認できるプログラム ID |

### 動作

注文完了ページに Felmat のコンバージョントラッキングタグが自動出力されます。

---

## 注意事項

- アフィリエイト連携は日本円（JPY）での取引にのみ機能します。
- 各アフィリエイトサービスの利用規約・計測タグの仕様変更により、動作が変わる場合があります。最新の仕様は各サービスの管理画面・ドキュメントを確認してください。
- タグが正しく発火しているかは、各アフィリエイトサービスのレポート機能やブラウザの開発者ツールで確認してください。
