# 郵便局振込（ゆうちょ振替）

[← ドキュメントトップ](index.md)

---

## 概要

郵便局（ゆうちょ銀行）の振替口座（郵便振替）による支払い方法です。銀行振込とは独立したゲートウェイとして提供されており、郵便局振替に特有の「記号」と「番号」形式の口座情報に対応しています。

---

## 有効化

**WooCommerce → JP4WC Settings → 決済設定 (Payment)** の「郵便局振込」セクションで有効にします。

または、**WooCommerce → 設定 → 決済** から「郵便局振込（JP4WC）」を選択して設定することもできます。

---

## 設定項目

| 設定項目 | 説明 |
|----------|------|
| 有効化 | 決済方法を有効/無効にします |
| 表示名 | チェックアウト画面での決済方法名（例: 郵便局振込） |
| 説明文 | チェックアウト画面での説明（振替手順など） |
| 記号 | 振替口座の記号（5桁） |
| 加入者番号 | 振替口座の番号（加入者番号） |
| 加入者名 | 振替口座の名義（カタカナ） |
| 振替案内文 | サンクスページ・メールに表示する案内文 |
| 振替情報表示位置 | 振替先情報を表示する場所 |
| 表示順序 | 振替情報と説明文の表示順序 |

---

## 顧客への表示

### チェックアウト画面

決済方法の選択肢として「郵便局振込」が表示されます。選択すると設定した説明文が表示されます。

### サンクスページ（注文完了ページ）

注文完了後、設定した郵便振替口座情報が表示されます。表示内容：

- 記号
- 加入者番号
- 加入者名
- 振替案内文

### 注文確認メール

顧客宛の注文確認メールに郵便振替口座情報が自動的に含まれます。

---

## ブロックチェックアウト対応

WooCommerce ブロックチェックアウトにも対応しており、追加設定なしで利用できます。

---

## 運用フロー

1. 顧客がチェックアウトで「郵便局振込」を選択して注文
2. 注文ステータスが「保留中」に設定される
3. 顧客が郵便局窓口・ATM または ゆうちょダイレクトから振替
4. 店舗側で入金を確認し、注文ステータスを「処理中」に変更
