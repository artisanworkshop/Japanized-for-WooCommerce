# 銀行振込

[← ドキュメントトップ](index.md)

---

## 概要

国内銀行への振込による支払い方法です。注文後に顧客へ振込先情報を表示・メール送信し、入金確認後に注文処理を進めるフローに対応します。

WooCommerce 標準の銀行振込（BACS）とは独立した、日本の銀行口座情報に特化した決済ゲートウェイです。

---

## 有効化

**WooCommerce → JP4WC Settings → 決済設定 (Payment)** の「銀行振込」セクションで有効にします。

または、**WooCommerce → 設定 → 決済** から「銀行振込（JP4WC）」を選択して設定することもできます。

---

## 設定項目

| 設定項目 | 説明 |
|----------|------|
| 有効化 | 決済方法を有効/無効にします |
| 表示名 | チェックアウト画面での決済方法名（例: 銀行振込） |
| 説明文 | チェックアウト画面での説明（振込手順など） |
| 銀行名 | 振込先の銀行名（例: ○○銀行） |
| 支店名 | 振込先の支店名（例: △△支店） |
| 口座種別 | 普通・当座など |
| 口座番号 | 振込先口座番号 |
| 口座名義 | 振込先口座名義（カタカナ） |
| 振込案内文 | サンクスページ・メールに表示する案内文 |
| 振込情報表示位置 | 振込先情報を表示する場所 |
| 表示順序 | 振込情報と説明文の表示順序 |

---

## 顧客への表示

### チェックアウト画面

決済方法の選択肢として「銀行振込」が表示されます。選択すると設定した説明文が表示されます。

### サンクスページ（注文完了ページ）

注文完了後、設定した振込先情報が表示されます。表示内容：

- 銀行名・支店名
- 口座種別・口座番号
- 口座名義
- 振込案内文

### 注文確認メール

顧客宛の注文確認メールに振込先情報が自動的に含まれます。

---

## ブロックチェックアウト対応

WooCommerce ブロックチェックアウトにも対応しており、追加設定なしで利用できます。

---

## 運用フロー

1. 顧客がチェックアウトで「銀行振込」を選択して注文
2. 注文ステータスが「保留中」に設定される
3. 顧客がサンクスページ・メールの振込先に入金
4. 店舗側で入金を確認し、注文ステータスを「処理中」に変更
