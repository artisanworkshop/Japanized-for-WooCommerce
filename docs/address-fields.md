# 住所フィールド・読み仮名・郵便番号自動入力

[← ドキュメントトップ](index.md)

---

## 概要

日本のネットショップに必要な住所フォームのカスタマイズ機能を提供します。読み仮名（ふりがな）フィールドの追加、郵便番号からの住所自動入力、敬称（様）の付与など、日本独自の慣習に対応します。

---

## 読み仮名（ふりがな）フィールド

### 機能概要

チェックアウトの請求先・配送先住所フォームに、姓・名それぞれの読み仮名フィールドを追加します。

- **姓（セイ）**: 姓の読み仮名
- **名（メイ）**: 名の読み仮名

入力された読み仮名は注文データに保存され、管理画面の注文詳細・顧客情報ページで確認できます。

### 設定方法

**WooCommerce → JP4WC Settings → 一般設定 (General)** を開きます。

| 設定項目 | 説明 |
|----------|------|
| 読み仮名フィールド | チェックをオンにすると読み仮名フィールドを表示します |
| 読み仮名必須入力 | チェックをオンにすると読み仮名を入力必須にします |

### ブロックチェックアウトでの表示

WooCommerce ブロックチェックアウト使用時も、読み仮名フィールドは姓・名フィールドの直後に自動で配置されます（CSS `order` プロパティによるレイアウト制御）。

---

## 敬称（様）

### 機能概要

顧客の姓または名の後ろに「様」を自動的に付与します。注文確認メールや管理画面上の宛名表示に反映されます。

### 設定方法

**WooCommerce → JP4WC Settings → 一般設定 (General)** の「敬称（様）」をオンにします。

---

## 会社名フィールド

### 機能概要

チェックアウトフォームの会社名フィールドの表示・非表示を制御します。

### 設定方法

**WooCommerce → JP4WC Settings → 一般設定 (General)** の「会社名フィールド」で表示/非表示を切り替えます。

---

## 郵便番号→住所自動入力（Yahoo! API 連携）

### 機能概要

チェックアウトおよびマイアカウントの住所編集ページで郵便番号を入力すると、都道府県・市区町村・番地が自動的に入力される機能です。

Yahoo! ジャパンの郵便番号検索 API を利用しています。

### 設定方法

1. [Yahoo! デベロッパーネットワーク](https://developer.yahoo.co.jp/) でアプリケーションを登録し、クライアント ID（アプリケーション ID）を取得します。
2. **WooCommerce → JP4WC Settings → 一般設定 (General)** を開きます。
3. 「郵便番号→住所自動入力」をオンにし、取得した **Yahoo! API キー（アプリケーション ID）** を入力して保存します。

| 設定項目 | 説明 |
|----------|------|
| 郵便番号→住所自動入力 | 機能の有効/無効 |
| Yahoo! API キー | Yahoo! ジャパンのアプリケーション ID |

### 動作

- 郵便番号入力フィールドからフォーカスが外れると自動的に住所を取得します。
- 取得した住所は都道府県・市区町村・番地1 の各フィールドに自動入力されます。
- API の取得は WordPress の REST エンドポイント（`/wp-json/yahoo/v1/postcode/`）経由で行われます。

## 住所フィールドの表示制御

### 仮想商品のみのカートでフィールドを非表示

ダウンロード商品やデジタルコンテンツなど、仮想商品のみのカートの場合、不要な住所フィールドを自動的に非表示にできます。

非表示にできるフィールド：

| フィールド | 設定キー |
|------------|----------|
| 郵便番号 | 請求先郵便番号 |
| 都道府県 | 請求先都道府県 |
| 市区町村 | 請求先市区町村 |
| 番地 | 請求先番地 |
| 補足住所 | 請求先補足 |
| 電話番号 | 請求先電話番号 |

**WooCommerce → JP4WC Settings → 一般設定 (General)** の各フィールドの表示設定で制御します。

---

## マイアカウントへの対応

読み仮名フィールドはマイアカウントの「住所」ページでも表示・編集できます。顧客が登録した読み仮名情報は次回チェックアウト時に自動で反映されます。
