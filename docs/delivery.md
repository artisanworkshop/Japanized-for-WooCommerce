# 配送日時指定

[← ドキュメントトップ](index.md)

---

## 概要

チェックアウト時に顧客が配送希望日・配送希望時間帯を指定できる機能です。配送不可曜日や祝日・休業期間を設定して、実際の配送能力に合わせた日付の選択肢を提供できます。

---

## 設定画面

**WooCommerce → JP4WC Settings → 配送設定 (Shipment)** から設定します。

---

## 配送日指定

### 基本設定

| 設定項目 | 説明 | デフォルト |
|----------|------|------------|
| 配送日指定機能 | 機能の有効/無効 | 無効 |
| 配送日必須入力 | 配送日を入力必須にする | 無効 |
| 最短配送日（日数） | 注文日から何日後から指定可能か | 2日後 |
| 最長指定可能日（日数） | 注文日から何日先まで指定可能か | 30日後 |
| 「指定なし」オプション | 日付未指定を選択可能にする | 有効 |
| 配送受付締め切り時刻 | この時刻以降の注文は翌日扱い（例: 18） | なし |
| 日付フォーマット | 表示する日付形式（例: Y/m/d） | Y/m/d |
| 曜日表示 | 日付に曜日を表示する（例: 月曜） | 無効 |

### 配送不可曜日

配送を行わない曜日にチェックを入れると、その曜日は選択肢から除外されます。

| チェックボックス | 対象曜日 |
|------------------|----------|
| 月曜日を除外 | 月曜日 |
| 火曜日を除外 | 火曜日 |
| 水曜日を除外 | 水曜日 |
| 木曜日を除外 | 木曜日 |
| 金曜日を除外 | 金曜日 |
| 土曜日を除外 | 土曜日 |
| 日曜日を除外 | 日曜日 |

### 祝日・休業期間

配送を行わない連続した期間（年末年始・お盆など）を設定できます。

| 設定項目 | 説明 |
|----------|------|
| 休業開始日 | 休業開始の日付（YYYY-MM-DD 形式） |
| 休業終了日 | 休業終了の日付（YYYY-MM-DD 形式） |

設定した休業期間中の日付は配送日の選択肢から自動的に除外されます。

---

## 配送時間帯指定

### 基本設定

| 設定項目 | 説明 |
|----------|------|
| 配送時間帯機能 | 時間帯指定の有効/無効 |
| 時間帯必須入力 | 時間帯を入力必須にする |
| 「指定なし」オプション | 時間帯未指定を選択可能にする |

### 時間帯の設定

配送時間帯の選択肢は管理画面から自由に追加・編集できます。一般的な設定例：

- 午前中（12時まで）
- 14時〜16時
- 16時〜18時
- 18時〜20時
- 19時〜21時

---

## チェックアウトでの表示

### 従来のチェックアウト（ショートコード）

配送日・配送時間帯フィールドは注文情報セクションに表示されます。

### ブロックチェックアウト

WooCommerce ブロックチェックアウト使用時、配送日・配送時間帯フィールドは「注文情報（Order information）」セクションに表示されます（`location: 'order'`）。

---

## 注文情報への反映

顧客が指定した配送日時は以下の場所に表示されます。

- **注文確認ページ（サンクスページ）**: 注文の概要に配送希望日・時間帯が表示されます。
- **注文確認メール**: 顧客・管理者宛のメールに配送情報が含まれます。
- **管理画面の注文詳細ページ**: 注文メタ情報として表示・編集できます。
- **マイアカウントの注文履歴**: 過去の注文に配送情報が表示されます。

---

## 設定例：通常のECサイト向け

```
配送日指定機能: 有効
最短配送日: 2日後
最長指定可能日: 30日後
配送不可曜日: 日曜日
「指定なし」オプション: 有効
時間帯: 午前中 / 14〜16時 / 16〜18時 / 18〜20時
```
